# Screenshots du README

Dépose ici les captures référencées par le README principal. Noms attendus :

## Côté étudiant
- `maze.png` — vue principale du jeu (labyrinthe + éditeur de code)
- `docs.png` — page documentation `/docs` (fonctions + badges/rangs/maîtrise)
- `tutorial.png` — tutoriel interactif au premier lancement

## Côté admin
- `progress.png` — page `/progress` (suivi d'avancement)
- `live-dashboard.png` — page `/live` (vue temps réel de la classe)
- `solutions.png` — page `/solutions` (explorateur de solutions et feedbacks IA)
- `level-editor.png` — page `/editor` (éditeur de niveaux)
- `similarity.png` — détection de similarité (constellation + diff)

## Format conseillé

- PNG, largeur 1200–1600 px, compressé (TinyPNG ou équivalent) pour rester sous ~300 Ko par image.
- Capture sur thème clair *ou* sombre — cohérent entre toutes les images.
- Anonymiser : pas de vraies identités d'étudiants (renomme en `Étudiant 1`, `Étudiant 2`, … si besoin).
