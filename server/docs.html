Documentation des fonctions

Toutes les fonctions disponibles pour piloter ton robot dans AlgoMaze.

🚶 Déplacement

moveForward()
action

Fait avancer le robot d'une case dans la direction actuelle. Si un mur lui fait face, le robot ne bouge pas (mais ta solution est considérée comme incorrecte au moment de la validation).

turnLeft()
action

Fait tourner le robot de 90° vers la gauche, sur place. Pour tourner à droite, tu peux enchaîner trois turnLeft().

💎 Gemmes

collectGem()
action

Ramasse la gemme située sur la case actuelle du robot. Sans effet s'il n'y a pas de gemme à cet endroit.

canCollectGem()
booléen

Renvoie true si une gemme non ramassée se trouve sur la case actuelle, false sinon.

🔘 Interrupteurs

toggleSwitch()
action

Bascule l'interrupteur situé sur la case actuelle (allumé ↔ éteint). Sans effet s'il n'y a pas d'interrupteur ici.

isOnSwitch()
booléen

Renvoie true si le robot est positionné sur un interrupteur (allumé ou éteint).

canActivateSwitch()
booléen

Renvoie true si la case actuelle contient un interrupteur éteint (qui peut donc être allumé).

canDeactivateSwitch()
booléen

Renvoie true si la case actuelle contient un interrupteur allumé (qui peut donc être éteint).

🧱 Détection d'obstacles

isBlocked()
booléen

Renvoie true si la case devant le robot (dans sa direction actuelle) est un mur ou hors du labyrinthe.

isBlockedLeft()
booléen

Renvoie true si la case à gauche du robot (relative à sa direction actuelle) est un mur.

isBlockedRight()
booléen

Renvoie true si la case à droite du robot (relative à sa direction actuelle) est un mur.

🏅 Badges par niveau

À chaque niveau validé, l'IA évalue ta solution et lui attribue un badge selon son score (0–100). Seul ton meilleur résultat est conservé : refaire un niveau ne peut jamais faire baisser ton badge.

Platine200 XP

Score ≥ 80 et contraintes pédagogiques respectées. La solution est élégante, efficace et met en œuvre les concepts attendus du niveau.

Or100 XP

Score ≥ 60 et contraintes respectées. Solution correcte et de bonne qualité.

Argent50 XP

Score ≥ 40 et contraintes respectées. Solution fonctionnelle mais perfectible.

Bronze25 XP

Score ≥ 20. Plafond maximum si les contraintes pédagogiques ne sont pas respectées (solution qui fonctionne mais n'utilise pas les concepts attendus).

Fer10 XP

Score ≥ 0. La solution arrive à la sortie mais présente des lacunes importantes.

🎖️ Rangs globaux

Le rang reflète la qualité récente de tes solutions. Il est calculé à partir de la médiane glissante des XP obtenus sur tes 10 derniers niveaux validés — pas une moyenne sur l'ensemble du parcours, ce qui signifie que progresser sur les niveaux récents peut faire monter ton rang même si tes premiers niveaux étaient faibles.

Légende

Médiane ≥ 150 XP — tu combines avancement ET qualité Platine de façon constante.

Maître

Médiane ≥ 100 XP — tes solutions récentes sont globalement de niveau Or ou Platine.

Expert

Médiane ≥ 60 XP — tu réussis bien mais avec quelques solutions moins abouties.

Initié

Médiane ≥ 30 XP — tu progresses, les bases sont là.

Apprenti

Médiane ≥ 0 XP — point de départ de tout le monde. Continue à t'entraîner !

📊 Niveau de maîtrise global

Le niveau de maîtrise affiché dans l'en-tête combine ton avancement dans le parcours et la qualité de tes solutions. Il est calculé ainsi :

Score de maîtrise = (XP total gagné) / (42 niveaux × 200 XP) × 100

Ainsi, compléter plus de niveaux et obtenir de meilleurs badges font tous les deux monter le score. Il ne peut jamais baisser.

Très bonne maîtrise

Score ≥ 70 % — tu as validé une grande partie du parcours avec une qualité élevée.

Maîtrise satisfaisante

Score ≥ 40 % — bon équilibre entre avancement et qualité.

Maîtrise fragile

Score ≥ 20 % — tu as démarré mais il reste beaucoup de marge de progression.

Maîtrise insuffisante

Score < 20 % — continue, chaque niveau validé fait monter le score !

L'en-tête affiche également ton avancement (Niv. X/42) et ton rang (basé sur la médiane des 10 derniers niveaux), mis à jour après chaque évaluation IA.

💡 Astuces JavaScript

Boucles

Pour répéter une action : for (let i = 0; i < 3; i++) { moveForward(); } ou while (!isBlocked()) { moveForward(); }.

Conditions

Pour réagir à l'environnement : if (canCollectGem()) { collectGem(); } ou if (isBlocked()) { turnLeft(); }.

Fonctions

Pour factoriser une suite d'actions, définis tes propres fonctions et appelle-les :
function tournerADroite() { turnLeft(); turnLeft(); turnLeft(); }
